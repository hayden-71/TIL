## SQL (Structured Query Language) 
1일차: 처음 보는 언어들의 향연.. 구조 만들고 데이터 CRUD!

전화번호 저장 시에 INT 대신 VARCHAR 사용이 낫다.

국룰: 정해진 명령어-대문자, 내가 정한 이름-소문자

- SQL은 데이터베이스에 저장된 데이터를 관리하고 조작하며, 데이터베이스 시스템 자체를 제어하기 위해 설계된 특수 목적의 프로그래밍 언어예요. 데이터베이스와 대화하기 위한 언어.

- 세미콜론(;): 각 SQL 문장의 끝에는 반드시 세미콜론을 붙여서 문장을 구분해야 한다.
- **Ctrl+Enter**: 한 줄씩 실행
